# Schedule Builder (Next.js)

Веб-приложение для разбора Excel-расписания, правки событий, визуализации таймлайна (пакеты дней, параллельные слоты) и экспорта: **печать / PDF в браузере** и **HTML+CSS для Тильды**.

Подробнее: **[документация в каталоге `docs/`](docs/README.md)**.  
Установка после клонирования с GitHub (**Docker** или **Node**): **[`docs/install.md`](docs/install.md)**.

## Возможности (кратко)

- Регистрация и вход (cookie-сессия `iron-session`)
- Проекты и версии сборок (**build**) из загруженных `.xlsx`
- Вкладки проекта: документы, события, архитектура таймлайна, техрасписание, аудитории, экспорт
- Данные на диске: `data/store.json` и файлы загрузок под `data/users/…`

## Требования

- **Node.js** LTS (рекомендуется текущая LTS-ветка, совместимая с Next.js 15)

## Быстрый старт

```powershell
cd "путь\к\schedule_builder_next"
npm install
npm run dev
```

Откройте `http://localhost:3000` — далее регистрация или вход.

### Скрипты

| Команда        | Назначение              |
|----------------|-------------------------|
| `npm run dev`  | Разработка (hot reload) |
| `npm run build`| Продакшен-сборка        |
| `npm start`    | Запуск собранного приложения (`next start`) |

## Структура репозитория (кратко)

| Путь | Назначение |
|------|------------|
| `app/` | App Router: страницы, layout, API-роуты |
| `lib/store.ts` | Пользователи, проекты, сборки, JSON-хранилище |
| `lib/excel.ts` | Разбор `.xlsx` |
| `lib/schedule.ts` | Дни, интервалы, раскладка по колонкам (lanes) |
| `lib/export-tilda.ts` | Генерация HTML/CSS для блока T123 |
| `lib/session.ts` | Сессия и `SB_SECRET` |
| `lib/user-files.ts` | Пути к каталогам загрузок |
| `data/` | Создаётся автоматически: `store.json` и пользовательские файлы |

Детали — в **[`docs/architecture.md`](docs/architecture.md)**.
